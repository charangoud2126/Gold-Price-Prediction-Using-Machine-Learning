# Gold-Price-Prediction-Using-Machine-Learning
Gold Price Prediction using Machine Learning | EDA | Random Forest | Python

## Project Overview
This project predicts gold prices using machine learning techniques based on financial indicators such as stock index, oil price, silver price, and currency exchange rate.

## Dataset Features
- SPX – Stock market index
- USO – Oil price
- SLV – Silver price
- EUR/USD – Currency exchange rate
- GLD – Gold price (Target)

## Steps Performed
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Correlation heatmap
- Model training using Random Forest
- Model evaluation

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
